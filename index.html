Java / Spring Boot TypeScript / Node.js Python APIs REST / SQL / Hibernate

Audit d'Architecture CTO as a Service Agile SCRUM / Management Revue de Code

DevOps & Intégration Continue

Jenkins / Git Linux (Ubuntu/CentOS) Docker / DevOps CI-CD / Cloud AWS
